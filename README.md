##  Features

- **Modern Design** - Futuristic dark theme with cyan/violet accents
- **Responsive** - Fully optimized for desktop, tablet, and mobile devices
- **Performance Optimized** - Lazy loading, deferred CSS/JS, caching headers
- **Interactive Elements** - Swiper.js carousel, animated skill bars, smooth scrolling
- **Frosted Glass UI** - Modern glassmorphism navbar effect
- **Contact Form** - Integrated with Formspree for email submissions
- **SEO Optimized** - Open Graph tags, meta descriptions, semantic HTML

## Deployment

This site is deployed on **Netlify** with automatic deployments from the `main` branch.

### Security Headers

The site includes security headers configured in `netlify.toml`:
- X-Frame-Options
- X-XSS-Protection
- X-Content-Type-Options
- Referrer-Policy
- Permissions-Policy
- Content-Security-Policy
